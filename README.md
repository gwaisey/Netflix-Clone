# 🎬 Netflix Clone - Laravel 11

A simple yet elegant Netflix Clone web application built with **Laravel 11**.

## 🚀 Features
- **Email-based Authentication**: Fast sign-in process that automatically registers new emails.
- **Movie CRUD**: Full functionality to Create, Read, Update, and Delete movie data.
- **Localization**: Support for both English and Indonesian languages.
- **Responsive UI**: Dark-themed interface inspired by the official Netflix design using Bootstrap 5.

## 🛠️ Tech Stack
- **Framework**: Laravel 11
- **Database**: SQLite
- **Frontend**: Blade Templating, Bootstrap 5, Google Fonts (Roboto)
- **Language**: PHP 8.2+
